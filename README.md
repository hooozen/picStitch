🎶 picStitch 没有后台服务器，加载完毕后离线可用.  不必担心数据安全和隐私问题.

picStitch 是为了解决手机系统图库不支持图片拼接，以及部分手机 (例如 iPhone，截止到 2023.9.2) 不支持长截屏.  同时第三方拼图软件过于臃肿或者广告较多的问题.
